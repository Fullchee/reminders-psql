# [Reminders backend](https://fullchee-values-backend.herokuapp.com/)

## Building the repo

```sh
npm run build
```

## Type-checking the repo

```sh
npm run type-check
```

And to run in `--watch` mode:

```sh
npm run type-check:watch
```

Develop

```sh
yarn dev
```
